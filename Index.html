Formatting Commands and Styles

Commands

Command Desc.
NEWLINE Next word starts on a new line
SKIPLINE Creates a full empty line, Next word starts on the following line
PAGEFILL Fills the rest of the lines


You can use these commands between words to add dynamic formatting to your text. You can go to a new line, skip a line, or skip an entire page!

Formatting

Property Desc.
#t: Text for your word
#f: Font for the word
#c: Color of the text
#bg: Color of the cell for the word
#p: Use a preset for formatting
#h: Link to URL or Index of new text value


You can add formatting to any word you want!

  • If you want to add some formatting then you need to add a #t: tag infront of whatever text you want to print
  • Formatting is read left to right, meaning the right most command will override any previous identical commands
  • You can format a command to fill cells with a color
  • You can create singled colored cells by not including any text
  • the #h: format requries an additional sub format of either :l for URLs or :p
